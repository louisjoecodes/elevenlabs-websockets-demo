# How to use the ElevenLabs Websocket Text-to-Speech Demo with Word Highlighting

In this tutorial, you’ll learn how to create a real-time text-to-speech chatbot that highlights words as they are spoken. We’ll leverage the ElevenLabs WebSocket API, starting with a backend in Python using FastAPI, and connect it to a frontend built with Next.js.

If you want to jump straight to the finished repo you can find it [here](https://github.com/louisjoecodes/elevenlabs-websockets-demo).
